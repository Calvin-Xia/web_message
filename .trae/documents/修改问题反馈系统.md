# 问题反馈系统修改计划

## 修改内容

### 1. schema.sql
- 修改表名为 `issues`
- 字段修改：
  - `content` → `issue`（问题）
  - 新增 `isInformationPublic` TEXT（是否实名：yes/no）
  - 新增 `name` TEXT（姓名）
  - 新增 `student_id` TEXT（学号）
  - 新增 `isReport` TEXT（是否上报：yes/no）
  - 删除 `email` 字段

### 2. index.html
- 修改标题为"问题反馈系统"
- 表单字段修改：
  - "留言内容" → "问题"
  - 删除邮箱字段
  - 新增"是否实名"复选框（控制姓名和学号显示）
  - 新增"姓名"输入框（条件显示）
  - 新增"学号"输入框（条件显示）
  - 新增"是否上报"复选框
- 列表显示：仅显示问题和时间，不显示个人信息

### 3. index.js
- API 端点：`/api/messages` → `/api/issues`
- GET 请求：仅返回 `id, issue, created_at`
- POST 请求：
  - 接收新字段并验证
  - 实名时必填姓名和学号
  - 根据条件存储个人信息
